# CyberBin Template Intake

This file tracks public-ready template intent. Do not store local screenshot paths, paid template screenshots, or temporary user files here.

## Template 1: `pinboard`

- Status: implemented and public in v1.
- Style: bright yellow paper, deep blue ink, cream paper cards, blue outlines, offset shadows, mono metadata, handwritten emphasis, hand-drawn paperclip marks.
- Best for: youthful product strategy reports, startup team playbooks, AI technology tool launches, Codex/AI tool decks, SaaS products, growth retrospectives, course covers, and startup pitches.
- Not best for: government reports, state-owned enterprise brochures, formal corporate profiles, banking, legal, medical, insurance, luxury, real estate, and heavy industry decks unless the user explicitly asks for a young innovation sub-brand look.
- Must keep: yellow/blue palette, paper-card system, mono labels, paperclip marks, three-card detail pages, workflow cards, comparison table, metric cards, quote page, and final ask layout.
- Do not include: right-side screenshot navigation dots from reference websites, paid website assets, remote asset URLs, or copied screenshots.
